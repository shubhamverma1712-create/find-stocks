🔗 TradingView Webhook Setup
1
Apna Glitch URL copy karein
Glitch project ka naam apne URL mein replace karein:
https://YOUR-PROJECT-NAME.glitch.me/webhook
2
TradingView Alert JSON Message
Alert create karte waqt "Message" field mein ye daalo:
{ "symbol": "{{ticker}}", "signal": "BUY", "price": {{close}}, "sl": 0, "tp": 0, "tf": "{{interval}}", "change": 0 }
3
Signal types ke liye alag alerts banao
Har alert condition ke liye "signal" field change karo:
• BUY signal → "signal": "BUY"
• SELL signal → "signal": "SELL"
• Pre-Buy → "signal": "PRE-BUY"
• Pre-Sell → "signal": "PRE-SELL"
